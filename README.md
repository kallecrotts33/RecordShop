RecordShop documentation

Description
RecordShop is an example of a website, that sells vinyl records. RecordShop was made as a part of a school assignment. RecordShop has two parts: the website itself, for purchasing records and so on, and then an admin page, where an administrator can alter the contents of the database with CRUDL functionality.

Usage and testing
