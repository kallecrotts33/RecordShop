RecordShop documentation

Description
RecordShop is an example of a website, that sells vinyl records. RecordShop was made as a part of a school assignment. RecordShop has two parts: the website itself, for purchasing records and so on, and then an admin page, where an administrator can alter the contents of the database with full CRUDL functionality.

Usage and testing

1. Initializing database: run command: 'RecordShop/webshop npm run db:init'
2. Starting webapp and admin page: 'RecordShop/admin npm run dev' & 'RecordShop/webshop npm run dev'
3. Accessing website: http://localhost:3000/
4. accessing Swagger: http://localhost:3000/docs
